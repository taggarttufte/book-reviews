# Book Reviews

Live site: **[taggarttufte.github.io/book-reviews](https://taggarttufte.github.io/book-reviews/)**

Long-form reviews on AI safety, philosophy of science, and epistemology. Each review argues something — what the book changed, where it falls short, and how it connects to alignment or my own research — rather than summarizing.

Built with Jekyll + `pages-themes/minimal` and a custom `reviews` collection (structured frontmatter: rating, tags, author, finished/reviewed dates). MathJax via kramdown.

## Adding a review

Drop a new file in `_reviews/` using the frontmatter template shown in an existing review (e.g. `_reviews/2026-04-18-empires-of-ai.md`), commit, and push. GitHub Pages rebuilds in ~60s.
